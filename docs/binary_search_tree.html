The keys in a binary search tree are always stored in such a way as to satisfy the binary-search-tree property: Let \(x\) be a node in a binary search tree. If \(y\) is a node in the left subtree of \(x\), then \(y.key\leq x.key\). If \(y\) is a node in the right subtree of \(x\), then \(y.key\geq x.key\).
